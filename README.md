## Cómo activar GitHub Pages

1. Ve a **Settings → Pages** en este repositorio
2. En "Branch" selecciona `main` y carpeta `/ (root)`
3. Guarda — en 2 min tu URL pública estará lista

## Flujo RSVP → Mesas
Cuando un invitado confirma en `index.html`, su información se guarda automáticamente en `localStorage`. Al abrir `mesas.html` desde el mismo dispositivo/navegador, el invitado aparece ya en la lista lista para asignarle mesa.
